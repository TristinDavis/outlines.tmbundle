Outline Tools
(c) 2005 Eric Hsu, textmate@betterfilecabinet.com

This bundle processes simple text outlines (where the level is determined by the number of leading tabs).

For instance,
Coffee
Tea
        Black tea
        Green tea
                China
                Africa
Milk
is a canonical one from http://www.w3schools.com/html/tryit.asp?filename=tryhtml_nestedlists2

Anyway, this set of commands transforms tab-outlines into the following forms:
  1. (inline)
    1. Markdown
    2. Kwiki (and other wikis like PMWiki, the Textmate Wiki)
    3. HTML (both unordered and ordered)(yes, it validates as HTML 4.01 Transitional)
  2. (in other applications)
    1. OmniOutliner (selection shows up in OO as an imported OPML)
  3. (in another document)
    1. OPML (since we need the non-obvious header to make it real OPML)
Yes, this list was made with "Tab to HTML OL". If you don't like the types of OL that I use down the hierarchy, edit the script "t2ol.pl" and change the @type array.

The original reason this bundle was made was for the OmniOutliner command, since OO inexplicably lacks a 'paste tab outline as Outline' command.
